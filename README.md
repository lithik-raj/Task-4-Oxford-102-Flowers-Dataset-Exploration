# 🌸 Oxford-102 Flowers Dataset Exploration

## 📌 Project Overview

This project explores the Oxford-102 Flowers dataset to understand its structure and analyze important dataset characteristics such as the number of classes and image resolutions.

## 📂 Dataset Information

* Dataset: Oxford-102 Flowers
* Total Images: 8,189
* Total Classes: 102 flower categories

## 🔍 Analysis Performed

* Loaded and examined the dataset
* Analyzed the total number of flower classes
* Studied image resolution statistics
* Visualized class distribution
* Displayed sample flower images with their class labels

## 🛠️ Technologies Used

* Python
* Google Colab
* NumPy
* Matplotlib
* Pillow (PIL)
* SciPy
